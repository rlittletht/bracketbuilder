Automatic Updates

BracketBuilder is periodically updated to fix issues and to add new features. This add-in is hosted which means it is updated automatically whenever these updates are made. While the add-in is tested extensively before an update is made to the host, its always possible that an update breaks something or changes something in a way that disrupts the way you use it. If this happens, we're sorry for the inconvenience. Please let us know at red@traynrex.com!

Update Summary

Here's a summary of updates that have been made to the add-in:

2.0.0 (10/3/2024)
  • New Feature! Added Place Next and Place Remaining commands to the Bracket toolbar.
    These commands automatically place the next (or the remaining) game(s) in the best field slots. (Best is, of course, entirely subjective)
  • New Feature! Added a sheet so you can customize the field and day rules in the tournament.
